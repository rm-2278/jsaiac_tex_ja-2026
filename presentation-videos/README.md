# 動画作成用GIFファイル一覧

## 📁 presentation-videos/ フォルダ構成

### 🎯 subgoal-visualization/ (サブゴール可視化)
階層的世界モデルが学習過程でどのようなサブゴールを提示するかを可視化

- `early-stage-96k.gif` - 学習初期段階（96,000ステップ）
- `mid-stage-296k.gif` - 学習中期段階（296,000ステップ）  
- `late-stage-395k.gif` - 学習後期段階（395,000ステップ）

**使用場面**: 
- 「階層的計画が期待通り機能していない」ことを説明
- 上位層が長期サブゴール（青）ではなく短期サブゴール（赤）のみ提示

### 🤖 policy-behavior/ (方策行動)
エージェントの実際の行動パターンを可視化

- `early-policy-42k.gif` - 初期方策（42,000ステップ）
- `mid-policy-170k.gif` - 中期方策（170,000ステップ）
- `late-policy-368k.gif` - 後期方策（368,000ステップ）

**使用場面**:
- 「学習しているが単純な行動パターンのみ」を実証
- 階層的な複雑な戦略が学習されていないことを示す

### ⏱️ timeline-progression/ (時系列比較)
特定タイミングでの学習状況比較

- `very-early-10k.gif` - 超初期段階（10,000ステップ）
- `comparison-232k.gif` - 比較用中間段階（232,000ステップ）

**使用場面**:
- 学習の時間経過による変化（または変化の欠如）を説明
- 「期待される階層的学習が起こっていない」ことの証拠

## 🎬 動画での推奨使用方法

### 1. 導入部分（20秒）
- `very-early-10k.gif` で初期状態を説明
- 「階層的世界モデルは長期計画を学習することが期待される」

### 2. 問題提起部分（20秒）  
- `mid-stage-296k.gif` と `late-stage-395k.gif` を並べて表示
- 「しかし実際には...」

### 3. 詳細分析部分（60秒）
- `early-policy-42k.gif` → `late-policy-368k.gif` で方策の変化（の欠如）
- `early-stage-96k.gif` → `late-stage-395k.gif` でサブゴール提示の問題

### 4. 結論部分（20秒）
- `comparison-232k.gif` で最終的な状況を要約

## 💡 ナレーション例

「Visual Pinpad環境では、Hierosは学習初期段階（10k steps）から後期段階（395k steps）まで、期待される階層的なサブゴール提示を実現できませんでした。上位層は青色の長期ゴールではなく、赤色の短期ゴールのみを提示し続けています。これは階層的計画の根本的な課題を示しています。」
